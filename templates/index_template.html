Folketingsvalg Danmark 2022

Præsentation af resultater fra Folketingsvalget i 2022, og sammenligning med valget i 2019. Danmarkskortet viser landets inddeling i opstillingskredse, og via kort kategori elementerne til højre for kortet, vælges der hvilken type data kortet farvelægges med. Det er muligt at zoom ind og ud på kortet, at flytte kortet samt gendanne kortet til oprindelig størrelse (reset knappen).

Data

Geografisk data: Danmarks Adressers Web API som har en åben API, Danmarks Administrative Geografiske Inddeling.

Folketingsvalg: Danmarks Statistik der blandt andet har valgresultater tilgængelig som åben data på xml format.
